Maths For Kids

"Purpose of this application is to auto populate questions for kids"

This desktop application randomly generates two number and depending on the option selected,the choice selector can be multiplication,subtraction,sum and division.
the user can choose the difficulty by changing the mode option below the application.
in an event the user wishes to skip the question he can do it by pressing the "skip button".
scores will be check by the program if the user gets the question correct the score board will show an increment. User also can restart the game by pressing "restart button" under game settings.
