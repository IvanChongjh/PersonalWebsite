Trading Risk Calculator

Desktop Application for traders

This application was developed to enable Traders with scalping strategy to conveniently calculate the overall risk of each trade. Allowing them to know how well they have done throughout the day.

After inserting Values

After user have inserted values of the trading day, The program will calculate on golden rules of risking 1-3% per trade.

if the program detects user risking more than 1-3% per trade the program will alert the trader
